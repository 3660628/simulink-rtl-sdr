rtlsdr_find_devices

rtl-sdr devices attached to the host can be discovered using this function. The function returns a summary about connected devices containg the device index, the name and other useful information.

Usage:

>> rtlsdr_find_devices


Found 1 device(s):

Device Index 0
  Vendor:   Realtek
  Product:  RTL2838UHIDIR
  Serial:   00000001
  Gains [dB]: -1.0 1.5 4.0 6.5 9.0 11.5 14.0 16.5 19.0 21.5 24.0 29.0 34.0 42.0 43.0 45.0 47.0 49.0 
  Tuner:    E4000
